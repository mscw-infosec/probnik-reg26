# Загадка

До начала Второй магической войны, в дни, когда Философский Камень находился в "Хогвартсе", один нумеролог предложил Альбусу Дамблдору задачу,
которую можно было бы поставить на пути к Камню: испытуемому даются зашифрованные письмена, и ключ к ним, но не даётся алгоритм шифрования.

Нам удалось узнать о первой версии такого задания - её и представляем вам для решения. Известно, что в этой версии использовался шифр, известный как "Усовершенствованный стандарт шифрования". В ней использовался пароль `I_l0v3_vs0sh`.

- Рекомендуемые утилиты: OpenSSL.
- Цель работы: дешифрование флага.
- Итог работы: получение доступа к флагу.
- Критерий оценки: предоставление правильного флага.

# Решение

Перебираем все возможные к использованию в openssl вариации AES, находим нужную.

```sh
openssl enc -d -aes-256-cbc \
  -pass pass:I_l0v3_vs0sh \
  -in cipher.bin
```
