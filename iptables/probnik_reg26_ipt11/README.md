# Почти случай с CTF

Иван заметил странные подключения к своему серверу. Кто-то очень часто подключается с TOS(Type of Service) 0xe0 по протоколу UDP на порт 1983, при этом в каждом пакете есть подстрока "madoka". Ограничьте подобные подключения.

- Рекомендуемые утилиты: iptables.
- Цель работы: создание правил(а) для брандмауэра.
- Итог работы: цепочка правил для брандмауэра.
- Критерий оценки: предоставление правильного флага, полученного от проверяющей системы.

## Решение

В `man iptables-extensions` находим модуль для фильтрации по TOS и по строке. Важно просто не забыть все нужные опции)

Итоговое правило:
```
-A INPUT -p udp --dport 1983 -m tos --tos 0xe0 -m string --string madoka --algo kmp -j DROP
```
