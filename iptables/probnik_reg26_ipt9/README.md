# Логов хочется
Иван захотел подробно отследить сетевую активность по протоколу NTP. Напишите правило, которое будет сохранять лог для каждого запроса по протоколу NTP, в логе должна встречаться подстрока "VSOSH NTP". Лог должен создаваться только для трафика NTP.

*Важно: не используйте опцию, которая использует логи ядра (из-за особенностей ядра, невозможно сделать автоматическую проверку в этом случае); используйте немного другой способ (в нём для обработки логов используется userspace демон)*

*В редких случаях имеет смысл перезапускать инстанс после проверки неправильного решения*

- Рекомендуемые утилиты: iptables.
- Цель работы: создание правил(а) для брандмауэра.
- Итог работы: цепочка правил для брандмауэра.
- Критерий оценки: предоставление правильного флага, полученного от проверяющей системы.

## Решение
-A INPUT -p udp --dport 123 -j NFLOG --nflog-prefix "VSOSH NTP"
