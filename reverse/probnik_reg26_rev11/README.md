# Виртуальный миксер
Однажды, один мой преподаватель потребовал пояснять все аббревиатуры. В ответ на вопрос о том, как можно интерпретировать "ВМ", он ответил "Виртуальный Миксер"

## Решение
Приводим код в порядок и видим по факту простейшую виртуальную машину, которая умеет 1) добавлять 1 к текущему символу ввода; 2) вычитать 3; 3) делить нацело на 2; 4) переходить к следующей ячейке; 5) сравнивать все ячейки с 0 и выводить "good", если это так.

Переписываем раскодирование байткода из bz2 и ксора с 0x67, и применяем все операции на каждый символ в обратном порядке.
