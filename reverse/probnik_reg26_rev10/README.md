# Я ходил в школу!
Я узнал столько всего за 8 класс в школе, что придумал задачку...

## Решение
Программа ищёт индекс символа ввода в алфавите и подставляет этот индекс в полином второй степени. Полученные значения сравниваются с константой.

Для решения нужно сообразить, что один байт ввода преобразуется в 16битное число (2 байта), и не забыть про то, что числа в little-endian. Далее просто посчитать корень квадратного уравнения.
