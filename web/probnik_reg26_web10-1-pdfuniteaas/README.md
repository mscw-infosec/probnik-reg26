## Решение

В таске две уязвимости, NoSQLi и Shell injection.

NoSQLi (то, что используется монга) – можно зафигачить в жсоне `{"$ne":""}` вместо строки с паролем

Сложный флаг – чере command injection (там название ласт файла должно быть вида `;ls${IFS}>`, нам в хедерах возвращают UTF-7 закодированную команду для дебага) листим файлы и делаем cat. Возвращаемая пдф будет не очень пдфкой, но кому оно важно-то.
