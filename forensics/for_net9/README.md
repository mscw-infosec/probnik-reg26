# Тёмные данные

После окончания Второй Магической Войны оставшиеся в живых Пожиратели смерти были отправлены в Азкабан.
Для того, чтобы никто не мог связаться с ними, были введены в действие самые строгие меры безопасности, но часть мракоборцев подозревает, что кому-то из Пожирателей удалось пронести с собой в Азкабан маггловское устройство, которое, кажется, называется "пересдатчик".

С помощью специального заклинания мракоборцам удалось записать сетевую активность около Азкабана. Помогите мракоборцам подтвердить или опровергнуть их гипотезу - проанализируйте файл на наличие подозрительных сигналов.

- Рекомендуемые утилиты: Wireshark, Python.
- Цель работы: исследование дампа сетевого трафика.
- Итог работы: получение доступа к флагу.
- Критерий оценки: предоставление правильного флага.

## Решение

Открываем иерархию протоколов, видим что передавалось небольшое количество TCP -> Data.

Фильтруем по ним, находим сообщения `CIPHER:XOR`, `KEY: ...` и `DATA: ...`. Ксорим DATA с ключом и получаем флаг.
