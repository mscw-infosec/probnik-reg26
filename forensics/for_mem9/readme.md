# Тайна пожирателей

После Второй магической войны Пожирателей смерти заключили в Азкабан.  
Все их устройства также были конфискованы. Вам достался дамп памяти одного из компьютеров пожирателей, работавший на Windows. Ваша задача выяснить чем занимались на этом компьютере.

- Рекомендуемые утилиты: volatility3, Python, Bash.
- Цель работы: исследование дампа памяти операционной системы.
- Итог работы: получить доступ до 2 флагов.
- Критерий оценки: предоставление правильных флагов.
