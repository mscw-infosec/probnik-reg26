# Прощай, мир

В Хогвартсе использовался маггловский ПК под управлением Windows.  
Для работы с учебными иллюзиями требовались заклинания из платного пособия, но один из учеников решил пойти по пути тёмной магии и применил запрещённое заклинание `GoodByeWorld` для получения кода активации учебника.

Заклинание сработало неверно.  
Все файлы на рабочем столе перестали читаться.

Ваша задача это извлечь ключ из следов магии, оставшихся в памяти.

- Рекомендуемые утилиты: volatility3, Python, Bash.
- Цель работы: исследование дампа памяти операционной системы.
- Итог работы: получить доступ до 2 флагов.
- Критерий оценки: предоставление правильных флагов.
