# Туннель

Во время проведения экзаменов на школу чародейства и волшебства "Хогвартс" накладывается мощное заклинание, не позволяющее трафику студентов (за исключением трафика к серверам проведения экзаменов) покидать её пределы.

Мальчик Гарри Б., студент 1 курса "Хогвартса", нашёл способ обойти заклинание и передавать информацию за пределы "Хогвартса". Преподаватели смогли перехватить часть трафика ко внешним серверам - найдите спрятанную в ней информацию.

- Рекомендуемые утилиты: Wireshark, Python.
- Цель работы: исследование дампа сетевого трафика.
- Итог работы: получение доступа к флагу.
- Критерий оценки: предоставление правильного флага.

## Решение

Открываем иерархию протоколов, видим достаточно много DNS-запросов. Фильтруем по ним. Видим запросы вида `1.AAA====.vsosh.infosec.moscow`, в которых изменяется первое число и данные после него. Логично предположить, что число - номер пакета, а данные - кодировка из семейства Base. Перебором находим Base32.

Декодируем, переставляем в правильном порядке, получаем флаг.
