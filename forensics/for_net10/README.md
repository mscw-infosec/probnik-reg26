# Удостоверение

После окончания Второй Магической Войны Министерство Магии провело масштабную реорганизацию Департамента магического правопорядка. В рамках реформ, мракоборцам стали выдавать магические артефакты, которые работали на основе позаимствованного у магглов изобретения - сертификатов и ассиметричной криптографии.

Чтобы мракоборец потерял часть своей магической силы при увольнении, используется механизм отзыва сертификатов, также позаимствованный у магглов.

Тем не менее, во время штатной проверки артефактов своих подчиненных, дежурный мракоборец заметил необычную активность одного из них - по заявлению мракоборца, "артефакт пытался деактивировать себя, а затем активировался вновь". Нам удалось получить запись активности этого артефакта - изучите её и разберитесь в том, что произошло с артефактом.

- Рекомендуемые утилиты: Wireshark, Python.
- Цель работы: исследование дампа сетевого трафика.
- Итог работы: получение доступ к флагу.
- Критерий оценки: предоставление правильного флага.

## Решение

Открываем иерархию протоколов, видим что в протоколе OCSP передавался "Malformed packet".

Фильтруем по OCSP, переводим из hex в ASCII, читаем. Получаем что-то невразумительное. Находим пакет "Malformed packet", видим в нём `XOR_KEY`. Ксорим все пакеты OCSP дальше с этим ключом, получаем флаг.
